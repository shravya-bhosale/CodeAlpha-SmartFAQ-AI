# SmartFAQ AI

## Description

SmartFAQ AI is an intelligent FAQ chatbot built using Python, Streamlit, and Natural Language Processing (NLP). It uses TF-IDF Vectorization and Cosine Similarity to provide the most relevant answers to user questions through a simple and interactive interface.

## Features

- Answer frequently asked questions
- TF-IDF Vectorization
- Cosine Similarity Matching
- Interactive Streamlit interface
- Chat history support

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- NLTK

## How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
streamlit run app.py
```
